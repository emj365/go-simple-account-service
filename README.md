# go-simple-account-service

## Go Doc

https://godoc.org/github.com/emj365/go-simple-account-service

## Dependencies

### MacOS

```bash
brew install dep
dep ensure
```

### Ohter

```bash
curl https://raw.githubusercontent.com/golang/dep/master/install.sh | sh
dep ensure
```

## Run

```bash
go run main.go
```
